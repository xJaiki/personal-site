A dead game

You may asking "what is Keyforge?". Keyforge is a TCG (trading card game) that me and my friend started getting interesed in 2022. The particularity of this TCG is that unlike other ones like Yu-gi-oh or Magic, this is not a deck building based game. You buy a deck (36 cards) and that's your deck, you can't add or change cards, because every deck you buy is unique, because an algorithm create them to be unique.
